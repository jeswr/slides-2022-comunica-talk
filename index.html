Easy access to enriched RDF using LDflex + Comunica

7 September 2022

LDflex

  • A domain-specific language for querying Linked Data on the Web

Can write things like:
person.friends.firstName
to get a list of your friends

RDF Reasoning

  • Generating implicit facts from explicit data and rules

For instance the explicit data that
  • Ruben is a Person
  • Person is a subClass of Agent
Combined with the rule
  • ?a is a ?b and ?b is a subClass of ?c implies ?a is a ?c
Results in the fact that Ruben is an Agent
